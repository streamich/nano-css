prefixer

Auto-prefixes styles with vendor prefixes using inline-style-prefixer.

nano.put('.foo', {
  flex: 1,
});
.foo {
  -webkit-flex: 1;
  flex: 1;
}

Installation

import { addon as addonPrefixer } from 'nano-css/addon/prefixer';
addonPrefixer(nano);
