# tachyons

Adds [Tachyons](https://tachyons.io/) utility rules to the `snake` chain.

```jsx
<div className={nano.s.f4.i.b.strike.ttu.serif.measure.fl.wTwoThirds + ''}>
  Hello world
</div>
```

Installs the `snake` addon automatically.

## Installation

```js
import { addon as addonTachyons } from 'nano-css/addon/tachyons';
addonTachyons(nano);
```
